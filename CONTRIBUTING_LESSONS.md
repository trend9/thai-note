# คู่มือการเพิ่มและอัปเดตบทเรียน (Lessons Data Guide) 📚✨

เพื่อให้คุณทำการเพิ่มหรือแก้ไขบทเรียนภาษาญี่ปุ่นได้อย่างง่ายดายและปลอดภัยที่สุด ปัจจุบันเราได้เปลี่ยนระบบเป็น **"ระบบแยกไฟล์เดี่ยวรายหน่วยบทเรียน (Modular Standalone Files + Auto Compiler)"** เรียบร้อยแล้ว! 

คุณไม่จำเป็นต้องเข้าไปยุ่งหรือแก้ไขไฟล์ขนาดใหญ่อีกต่อไป เพียงแค่จัดเพิ่มไฟล์ `.json` หน้าใหม่เข้าไปในโฟลเดอร์ระบุ และระบบจะนำมาเชื่อมโยงกันอย่างสมบูรณ์แบบทันที!

---

## 📂 ตําแหน่งโฟลเดอร์ที่คุณต้องดูแลคัดกรอง
*   **โฟลเดอร์เก็บข้อมูลบทเรียนย่อย:** `/src/data/lessons/` (ประกอบด้วยไฟล์แยกเช่น `1.json`, `2.json`, ... ไปจนถึง `164.json` และคุณสามารถตั้งชื่อเพิ่มตัวเลขเข้าไปได้เลย เช่น `165.json`)

*ข้อมูลภายในโฟลเดอร์นี้จะถูกรวบรวม (Compile), เรียงตามรหัส `id` จากน้อยไปมากตัวเลข และสร้างออกมาเป็นไฟล์ `/src/data/lessons.json` อัตโนมัติทุกครั้งที่รันเซิร์ฟเวอร์ หรือเมื่อมีการสั่ง Build เว็บไซต์ขึ้นคลาวด์!*

## 💡 ประโยชน์และกฎเหล็กในการทำงาน (Best Practices)

1. **ปลอดภัยจาก Syntax JSON พังทั้งเว็บ:**
   * หากคุณเขียน commas `,` หรือวงเล็บปีกกา `{}` ตกหล่นในไฟล์บทเรียนอันใดอันหนึ่ง จะส่งผลลบแค่ไฟล์หน้าเดียวนั้น ไม่ทำให้โครงสร้างหลักของบทเรียนอื่นๆ พังเสียหาย 

2. **ระบบการจัดลำดับ (ID Ordering Sequence):**
   * ระบบ Compiler ส่วนหลังบ้าน (Backend Server) จะทำการเรียงลำดับบทเรียนขึ้นสารบัญอ้างอิงจากฟิลด์ตัวเลข `"id"` ดังนั้น คุณจึงตั้งชื่อไฟล์เป็นอะไรก็ได้ (เช่น `165.json`, `special-page.json`) ขอเพียงแค่ระบุระเบียบ `"id": 165` ไว้ภายในไฟล์ให้ถูกต้อง ระบบจะนำมาจับคู่เข้าตำแหน่งอย่างเหมาะสมเสมอ

3. **การทำงานของฟิลด์เสียงอ่านอัจฉริยะ (Dynamic Voice TTS):**
   * ฟิลด์ภาษาญี่ปุ่น `"phrase_ja"` จะถูกเรียกผ่าน API สังเคราะห์เสียงพูดอัตโนมัติบนตัวจำลองเมื่อเริ่มสัมผัสลำโพง 🔊 มีอัตราความถนัดแบบสำเนียงเกียวโต/โตเกียวดั้งเดิมที่คมชัด

4. **SEO & Sitemap Coverage:**
   * ทันทีที่มีการนำบทเรียนไปเรียบเรียงใหม่ ระบบสืบค้นข้อมูลหุ่นยนต์ภายนอก (Googlebot Crawlers) ที่มาเยือนหน้าเส้นทางด่วนพิเศษของเว็บไซต์คุณ จะค้นพบหน้าย่อยใหม่นี้ที่ถูกส่งลงในผัง `sitemap.xml` และแผ่นดินหน้าหลักเพื่อเริ่มดักลูกค้าแบบสิริมงคล!

## 🚀 ขั้นตอนการสร้างบทเรียนหน้าใหม่ (ง่ายและเร็วที่สุด)
1. เข้าไปที่โฟลเดอร์ `/src/data/lessons/`
2. เพื่อความเร็ว ให้คัดลอกไฟล์ที่มีอยู่แล้ว (เช่น `1.json`) แล้วเปลี่ยนชื่อไฟล์เป็นชื่อที่คุณต้องการ (เช่น `165.json`)
3. เปิดไฟลนั้นขึ้นมาแก้ไขเนื้อหาให้เป็นของบทเรียนใหม่ (รหัส `"id"` ห้ามซ้ำกันเด็ดขาด!)
4. เซฟบันทึกไฟล์ -- ระบบจะรวมเนื้อหา (Compile) สดใหม่ให้โดยระบบส่วนกลางทันที! 🥳
